# Pocket_PSU
A variable current &amp; voltage PSU with 3S lipo charging, balancing, and protection circuit using PPS. 

For times where you just wish you could carry a mini bench PSU in your pocket...

Safely charge from any USB C source, and delivery 2.8V - 22V at 0.12A - 3A.
